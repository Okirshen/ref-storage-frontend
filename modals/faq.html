FAQ

This is a storage site for reference images of rockets. The site is intented for use with sfs, 3d modelling or any other form of art. Just search for a reference or use the side bar to browse and download the reference you want.

First you need to be an authorised publisher. If your not already authorised and wish to be, dm @Rage Boy#8106 on discord.

After you are authorised you need to login with discord using the profile icon at the top right.

Then you can use the 'add ref +' button to add your reference to our collection. Your discord name and pfp will be associated with the reference labeled as the uploader.

You can use the feedback area to suggest a feature on the website or to provide feedback / critisism.

You can use it to suggest categories or renames but not for induvidual reference images.
